Detailed Description

template<typename Key, typename T, typename Hash = std::hash<Key>, typename KeyEqual = std::equal_to<Key>, typename Allocator = std::allocator<std::pair<const Key, T>>>
class aalbatross::utils::collection::SUMap< Key, T, Hash, KeyEqual, Allocator >

Streamable Unordered Map class extends STL Unordered Map and implements StreamableCollection interface.

Streamable UnorderedMap can be used to transform Map using map/reduce style operations on Stream.
