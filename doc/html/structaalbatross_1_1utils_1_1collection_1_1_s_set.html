Detailed Description

template<typename Key, typename Compare = std::less<Key>, typename Allocator = std::allocator<Key>>
class aalbatross::utils::collection::SSet< Key, Compare, Allocator >

Streamable Set class extends STL Set and implements StreamableCollection interface.

Streamable Set can be used to transform Set using map/reduce style operations on Stream.
