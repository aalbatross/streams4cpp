Detailed Description

Implementations of Collector that implement various useful reduction operations, such as accumulating elements into collections, summarizing elements according to various criteria, etc.

The following are examples of using the predefined collectors to perform common mutable reduction tasks:

// Accumulate names into a List
auto list = people.stream() .map([](auto people){return people.name;}) .collect(Collectors::toVector<std::string>());

// Accumulate names into a TreeSet
auto set = people.stream() .map([](auto people){return people.name;}) .collect(Collectors.toSet<std::string>());

// Convert elements to strings and concatenate them, separated by commas
auto joined = things.stream() .map([](auto thing){return toString(thing);}) .collect(Collectors::joining(", "));

// Compute sum of salaries of employee
auto total = employees.stream() .collect(Collectors::summingDouble([](auto emp){return emp.salary;}));

// Group employees by department
std::map<Department, std::vector<Employee>> byDept = employees.stream() .collect(Collectors.groupingBy<Employee>([](auto emp){return emp.dept;}));

// Compute sum of salaries by department
std::map<Department, double> totalByDept = employees.stream() .collect(Collectors.groupingBy<Employee>([](auto emp){return emp.dept;}, Collectors::summingDouble([](auto emp){return emp.salary;})));

// Partition students into passing and failing
std::map<bool, std::vector<Student>> passingFailing = students.stream() .collect(Collectors::partitioningBy<Student>([](auto student) {return s.getGrade() >= PASS_THRESHOLD}));
