Detailed Description

template<typename Key, typename T, typename Compare = std::less<Key>, typename Allocator = std::allocator<std::pair<const Key, T>>>
class aalbatross::utils::collection::SMap< Key, T, Compare, Allocator >

Streamable Map class extends STL Map and implements StreamableCollection interface.

Streamable Map can be used to transform underlying Map using map/reduce style operations on Stream.
